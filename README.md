# Simple Quiz App

A simple Flutter quiz application that tests your knowledge with a set of multiple-choice questions. It tracks your score and tells you exactly how many answers you wrote correctly out of the total.

## Features
- Multiple choice questions
- Tracks your correct answers (score)
- End-of-quiz results screen
- Restart quiz functionality
- Fully responsive across mobile, desktop, and web

## Getting Started

1. Clone or download the repository.
2. Ensure you have Flutter installed.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to start the app.
